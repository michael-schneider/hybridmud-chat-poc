Project Documentation

Tools used

The documentation uses a local version of GitBook. GitHub Wikis would have been an alternative, but the advantage of GitBook is not dependent on GitHub.

Usage for Development

The following scripts are defined in /package.json.

Installing the dependencies can be done with:

$ npm run docs:prepare

Building and serving the book on http://localhost:4000:

$ npm run docs:watch

Building from scratch:

$ npm run docs:build

Building the documentation and committing it to the gh-pagesbranch:

$ npm run docs:publish

Install in new Project
